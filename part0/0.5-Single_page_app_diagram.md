    server-->>browser: the css file
    deactivate server

    browser->>server: GET https://studies.cs.helsinki.fi/exampleapp/spa.js
    activate server
    server-->>browser: the SPA JavaScript file
    deactivate server

This diagram shows the initial process when a user visits the single-page app version of the notes app. The browser fetches the HTML document, the CSS file, and the SPA JavaScript file from the server.
